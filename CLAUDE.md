# MT5 Router

## Overview
Robust dashboard for managing MetaTrader 5 instances with VNC access, trading API, and real-time monitoring.

## Tech Stack
- Backend: Python FastAPI + Docker SDK + mt5linux
- Frontend: React 18 + TypeScript + Tailwind CSS
- Database: SQLite
- Container: Docker

## Quick Start
```bash
docker-compose up -d
```

Access: http://localhost:8080
Default credentials: admin / admin123
